[Jelsa]守护

2.

艾莎是一个很特别的小姑娘。她安静又害羞，有着不符合她年龄的沉稳，而她表现出来的敏感和小心翼翼杰克不曾在任何一个同年龄的孩子身上见到过。 

 

为了让她开心起来，杰克可着实费了不少功夫。他之前从未与孩子直面交谈过，所以很多时候的话语都显得很笨拙，但他身上有一种独特的亲和力，能让人觉得他是可以信任的。 

 

对于艾莎来说，远离父母妹妹已有一年的她渴望与别人交谈，亦是渴望有人陪伴的，所以她渐渐接受了杰克的存在，开始对他敞开心灵。 

 

与艾莎混熟后，杰克就经常来找她玩，艾莎那和自己相似的冰雪魔法让杰克感到稀奇不已。 

 

“所以，你也是被月亮唤醒的吗？” 

 

“我爸爸说我是天生的。” 

 

“人们都能看见你吗？我是说——有没有人会看不见？” 

 

“为什么会有人看不见我？” 

 

“这真是太神奇了……”杰克喃喃地说道，“几百年来，我第一次遇见和我有相同魔法属性的人，而你还不是月亮之神……” 

 

“月亮之神是什么？” 

 

“唔，就是我们这些被月亮唤醒的神，圣诞老人啊，牙仙啊，复活节兔子啊……”杰克掰着手指数道，“相信我们的孩子就能看见我们，不相信的就看不见。” 

 

“可我压根都不知道你的存在，还是看见你了呀。” 

 

“这个嘛……”杰克挠着下巴，慢慢地说道，“大概……是因为你也有魔法？” 

 

好吧，艾莎勉强接受了这个解释。 

 

// 

 

杰克很多时候都像个半大的孩子。月亮将他永久地定格在了少年时的状态，而几百年的岁月也没能改变他爱玩的个性。杰克长时间观察孩子的经验和相同属性的魔法使他们心意相通，和他在一块的时间里，艾莎没有一刻是感到枯燥的。 

 

杰克鼓励艾莎去释放自己的魔法。 

 

“这没什么可怕的，艾莎，你不用感到恐惧。”他轻声细语地说道，“嘿，看着我。我们为什么不来玩点有意思的呢？” 

 

杰克陪在她身边，慢慢地引导她来掌控自己的魔力，教她做出好看的冰雕。 

 

艾莎不敢出去，他就陪她在空旷的大房间里玩。艾莎的脚轻轻一跺，地板就覆盖上了冰面，而杰克就飞来飞去地用木杖去点墙壁，让四周充满了好看的冰花。 

 

时隔一年，艾莎再一次调动起体内的魔力，两手拢起的中心出现了旋转的雪球。她往上一抛，雪球在空中像烟花一样炸开，一瞬间房间里到处飘着亮闪闪的雪花。 

 

他们在房间里溜冰，堆雪人打雪仗，玩得不亦乐乎。杰克仗着别人看不见也听不见自己，就在空中大呼小叫，而几乎没有人经过的偏僻房间也能让艾莎尽情地笑闹。艾莎仿佛又回到了原来和安娜一起玩的时光，而这次她都无需担心自己会伤到杰克。 

 

玩累了，他们就坐在窗台上聊天。杰克给艾莎讲他几百年来单方面给予孩子们的欢乐，讲他和其他月亮之神之间的矛盾，而艾莎就给他讲自己的父母幼妹。他们有很多话题可以聊。 

 

// 

 

杰克有时候会有一些奇奇怪怪的念头。 

 

“你看，我们都有冰雪魔法，也勉强算是个同族了吧。以后我就是你的哥哥了，我会一直保护你的。” 

 

艾莎低头捣鼓着杰克的木杖，勉强“嗯”了一声，而杰克继续发散着自己的思维。 

 

“你有没有自己的神灵称号，比如——艾莎冻人什么的？” 

 

“艾莎冻人？”她重复了一遍这个称号，咯咯地笑了起来。 

 

“怎么样，是不是很棒？”杰克洋洋得意地说，“我是杰克冻人，你是艾莎冻人，我们就是冻人二人组。” 

 

“我不要，听起来可真怪。” 

 

// 

 

杰克是打心眼里喜欢这个小姑娘。 

 

“你比邦尼他们要可爱多了。”他一本正经地说着，“我见过那么多的孩子，你是其中最讨人喜欢的那一个。” 

 

杰克从不吝啬自己对这位小朋友的赞美，不要怕、真棒、乖女孩、小天使……在他的眼里，艾莎简直是世界上最好的姑娘。 

 

当艾莎对他诉说自己的心事，说她是多么渴望能和父母妹妹在一起时，他的心仿佛也随着她的落入了谷底。 

 

他心疼这么好的一个人类小姑娘，她思念家人，却只能独自禁锢在这房间里，不能和人接触，也不被人所知。这样的孤独日子他已经过了几百年了，但他不想让这个孩子也遭受这一切。 

 

习惯了四处游荡的杰克冻人，以天地为伴的杰克冻人，第一次有了这样的想法，他要守护这个孩子，让她找回失去的一切。
