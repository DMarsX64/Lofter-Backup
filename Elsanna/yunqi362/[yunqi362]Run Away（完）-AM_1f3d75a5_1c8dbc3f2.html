Run Away（完）

“－”是Anna独白

————————————————————————


We should be run away.


－你能听见我的呼唤吗？

 我总是执着地追求曾经失去的


－我一直注视着你

可我忘记我早就拥有更美好的一切


－在孤独的城堡中昏昏欲睡

所以上天惩罚我失去了你


－在梦中与你不分彼此地亲昵

我下定决心寻找你


－我开始沉下心等待

如同你从未放弃我


－这是一件苦差事——你知道我没什么耐心的

这一路上我十分“自由”——没人记得我是谁


－城堡开始变得不像我们曾经的家——时间冲刷了人们的记忆

孤独又试图吞噬我，所谓的自由是阿塔霍兰强加给我的空虚——可见我当时做了个多么愚蠢的决定


－我知道如果我想要再次与你相遇，就得离开曾经的阿伦戴尔

我知道如果我想要再次与你相遇，就得离开现在的阿塔霍兰



We must run away.


在去北山的途中遇见了你

－我跟随心的指引与你相遇


你依旧是你，无论如何我都会爱你

－我忘记了很多事，但从未忘记爱你


你在夏季描绘冬季

我们一起回忆

－冰封的记忆开始流动


不必惶恐忧虑，我一生所爱只有你

－与你相爱是我一生所向


铭记过去便让它过去

－否则会失去更多


不要忧虑我们曾经失去的

－我们已拥有彼此



现在

我们亲吻着

我们拥抱着


我们感受着彼此


不在冰冷的回忆之河

不在孤独的城堡之中


Now where we are stand.


When we together,we will know better.
